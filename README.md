# Macro-Geopolitical Dashboard

Interactive dashboard analyzing macro-economic indicators, gold prices, and geopolitical risk.

## Features
- Real-time commodity prices (Oil, Gold, USD Index)
- Inflation tracking
- Geopolitical risk analysis
- Correlation matrix
- Regression analysis
- Export data to CSV

## Tech Stack
- Python
- Streamlit
- Pandas, NumPy
- Plotly
- Scikit-learn

## How to Run
```bash
pip install -r requirements.txt
streamlit run dashboard.py
```
